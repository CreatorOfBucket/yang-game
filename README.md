# 🐏 羊了个羊 - 网页版

一款基于经典"羊了个羊"玩法的纯前端网页消除游戏，无需安装，打开浏览器即可游玩。

## 游戏截图

打开 `yang.html` 即可体验！

## 玩法说明

- 点击牌面上的卡牌，将其收入底部槽位
- 槽位中每凑齐 **3 张相同卡牌** 即可消除
- 槽位最多容纳 **7 张**，满了则游戏失败
- 通关后自动进入下一关，难度逐步递增

## 游戏特色

- **多关卡设计** — 从入门到地狱，共多个难度关卡
- **道具系统** — 撤回、洗牌等辅助道具
- **纯前端实现** — 单个 HTML 文件，无需后端，离线可玩
- **移动端适配** — 支持手机浏览器游玩

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/CreatorOfBucket/yang-game.git

# 直接用浏览器打开
open yang.html
```

或者直接下载 `yang.html` 文件，双击用浏览器打开即可。

## 技术栈

- HTML5 + CSS3 + 原生 JavaScript
- 无任何外部依赖（仅引用 Google Fonts 字体）
